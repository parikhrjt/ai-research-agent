# AI Research Agent — Sample Documents

This directory contains sample files for testing the ingestion pipeline.

| File | Format | Description |
|------|--------|-------------|
| `transformer_architecture.md` | Markdown | Notes on the "Attention Is All You Need" paper |
| `rag_pipeline_notes.txt` | Plain text | RAG system design notes |
| `experiment_results.csv` | CSV | Synthetic experiment metrics |
| `research_brief.pdf` | PDF | Generated research brief (text-based) |

Upload these via the Streamlit UI or `POST /upload` to populate the vector store.
